## 郑重声明

> 原`XxPay聚合支付`项目已更为Jeepay，由原XxPay团队开发维护。目前`Jeepay`开发已经进入测试阶段，最晚6月初会发布源码到Github和码云上。

> 新版Jeepay支付系统，使用SpringBoot + Ant Vue开发。适合互联网企业使用，支持普通商户和服务商模式接口，对接微信、支付宝、云闪付官方通道。支持主扫/被扫（聚合，通过接口下单指定金额）。

> 请关注官方网站www.jeequan.com或官方公众号（微信搜索：计全科技），获取Jeepay最新发布消息，如果您喜欢该项目，不妨随手Star下。

> 待发布源码时，更公布官方的qq或微信技术群，同时会发布全面的使用文档及演示地址供大家体验。
